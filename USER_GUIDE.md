### 4. Customize Colors

Choose your brand colors:

**Primary Color:**
- Used for main elements
- Typically your brand color
- Click the circular color picker
- Hex value shows beside it

**Secondary Color:**
- Complements primary
- Used for accents
- Should contrast with primary

**Tips:**
- Blue + Darker Blue: Professional
- Green + Teal: Modern/Tech
- Purple + Blue: Creative/Tech
- Orange + Red: Energy/Excitement

### 5. Upload Brand Image (Optional)

Add your logo or brand image:

**How to upload:**
1. Click the upload area
2. Select PNG, JPG, or GIF
3. Max 5MB file size
4. See "‚úì Image ready" when done

**Best practices:**
- 1:1 square images work best (500x500px minimum)
- Transparent PNGs are ideal
- Keep file size under 2MB for faster generation

### 6. Generate Your Design

Click the big blue "‚ú® Generate Design" button

**What happens:**
1. Button shows spinner
2. Text says "Generating Design..."
3. Takes 1-2 seconds
4. Design appears in preview on right

### 7. Download Your Design

Click the green "Download" button

**You'll get:**
- SVG file (vector format)
- Named with your company + design type
- Example: `TechVision-business_card.svg`

**SVG Benefits:**
- Scalable to any size
- Perfect for printing
- Can be edited in graphic design tools
- Very small file size

## ‚ùì Troubleshooting

### Form says "Required field missing"
**Problem:** Can't generate design
**Solution:** 
- Check which fields show as required (marked with *)
- Each design type has different requirements
- Don't worry about optional fields

### Image didn't upload
**Problem:** "Image must be less than 5MB"
**Solution:**
- File is too large
- Compress the image
- Use online tool: TinyPNG or ImageOptim
- Try again with smaller file

### Design looks same every time
**Problem:** Getting identical layouts
**Solution:**
- This shouldn't happen (should see 4 different styles)
- Try refreshing the page
- Clear browser cache
- Regenerate a few more times
- Each should be different layout

### Preview not showing
**Problem:** Preview area is blank
**Solution:**
- Check all required fields are filled
- Click Generate button
- Wait 2-3 seconds for generation
- Make sure no error message is showing

### Can't download file
**Problem:** Download button not working
**Solution:**
- Check that preview is showing
- Try a different browser
- Clear browser cache
- Generate design again
