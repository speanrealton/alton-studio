## Common Questions

### Q: Can I change the circle name URL?
A: No, the URL (slug) is fixed to preserve links. You can change the display name, but the URL stays the same.

### Q: How long do image changes take to appear?
A: Usually instantly, but may take 1-5 minutes due to caching. Images are cached for performance.

### Q: What happens if I upload a huge image?
A: It's automatically resized to optimal dimensions. No need to resize manually!

### Q: Can I use emojis in the name?
A: Yes! But they count toward the 50 character limit.

### Q: What if I upload the wrong image?
A: Simply click to upload again - it replaces the previous one.

### Q: Can non-members edit?
A: No, only the circle creator can edit. This protects your community.

### Q: What file types are supported?
A: All common image formats (JPG, PNG, GIF, WebP, etc.)

### Q: Is there an undo button?
A: No, but you can edit again anytime to change it back.

### Q: Can I make the circle private?
A: That setting was made during creation and cannot be changed. Create a new circle if needed.

### Q: Do I need to be a member to edit?
A: No, you automatically created a membership when you created the circle.

### Q: Can others see my edit history?
A: No, only current information is visible.

## Troubleshooting

### Image Upload Failed
- Try a different image format
- Ensure image file isn't corrupted
- Check file size (shouldn't be huge)
- Try again - might be temporary

### Save Didn't Work
- Check your internet connection
- Look for error message
- Try saving again
- Refresh and check if changes saved

### Changes Not Showing
- Refresh the page
- Clear browser cache (Ctrl+Shift+Delete)
- Wait 1-5 minutes for cache to update
- Try a different browser

### Can't Find Edit Button
- Make sure you created the circle
- Make sure you're logged in
- Only creators see the button
- Ask the creator if you're a member

### Form Won't Submit
- Fill in the circle name (required)
- Check for error messages
- Ensure images uploaded successfully
- Try again
